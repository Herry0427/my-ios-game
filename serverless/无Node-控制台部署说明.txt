在不安装 Node.js 的前提下，可以用 Cloudflare 网页控制台部署本目录的 Worker（逻辑与 cloudflare-worker-football.mjs 一致）。

1. 打开 https://dash.cloudflare.com 并登录
2. 左侧「Workers 和 Pages」→「创建」→「创建 Worker」
3. 用「Quick edit」或代码编辑器，把本目录文件 cloudflare-worker-football.mjs 的全部内容粘贴进去（覆盖默认代码）
4. 右上角「保存并部署」
5. 记下分配的 workers.dev 子域，你的赛程接口地址为：
     https://<子域>.workers.dev/football-bundle.json
6. 打开上级目录的 config.js，填写：
     window.__IOS_GAME_FOOTBALL_REMOTE__ = "https://<子域>.workers.dev/football-bundle.json";
7. （可选）若百度返回验证页：在 Worker → 设置 → 变量和机密 → 添加机密变量名 BAIDU_COOKIE，值为浏览器里登录百度后的 Cookie 字符串

说明：控制台部署后，日后改代码需在网页里再粘贴保存；有 Node 时可用 deploy_worker.bat 一条命令更新。

（同内容 ASCII 文件名副本：NO_NODE_DASHBOARD_DEPLOY.txt，便于脚本引用。）
